## 说明
目前市面上主流的 PHP - MVC 框架最低环境配置均为 php5.3，由于环境限制，为了方便使用 php5.2 进行开发，自建简易的 MVC 框架 VSF (victorsun framework)，该框架基于 `PHP5.2 ` 进行开发。
